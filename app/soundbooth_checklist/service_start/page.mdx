# Service Start

## OBS

<Steps>

    ### Prelude Music Begins
    When the prelude music starts, Click the ***Start Streaming*** button (the recording will automatically start when the stream starts)

    ### Service Begins
    When the service starts, select the ***Church Livestream*** scene in the ***Scenes*** section on the dashboard
    
</Steps>


## Audacity

Click the ***Record*** button

## Attendance

Set the ***Service Started*** time (`Win` + `Shift` + `;`)
