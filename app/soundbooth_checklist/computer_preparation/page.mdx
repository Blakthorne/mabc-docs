# Computer Preparation

## System Settings

### Send Audio to TVs
1. Click on the Sound Icon in the Taskbar
2. Set ***Sound Output*** to ***LCD TV***

## Audacity

<Steps>

    ### Prepare Recording Track
    1. Go to ***Desktop/***
    2. Open ***Temporary.aup3***
    3. Delete the track containing the last recording
    4. Go to ***Audio Setup*** and ensure these settings are used—
        - **Playback Device** - X-USB
        - **Recording Device** - X-USB
        - **Recording Channels** - 2 (Stereo)

    ### Set Metadata
    1. Click ***Edit***
    2. Click ***Metadata Editor*** and set Metadata with these settings—
        - **Artist Name** - \{ *Speaker Name* \}
        - **Track Title** - \{ *Sermon Title* \}
        - **Album Title** - Missouri Avenue Baptist Church Downloads
        - **Track Number** - \{ *Last Track Number + 1* \}
        - **Year** - \{ *Current Year* \}
        - **Genre** - Podcast
        - **Comments** - \{ *Verse Reference for Sermon* \}
    3. Click ***OK***

</Steps>

## Website

<Steps>

    ### Get Livestream Link
    1. Go to the ***Upcoming*** livestream in YouTube as outlined [here](#youtube)
    2. Copy the video link

    ### Add Sermon Page
    1. In Chrome, go to [Church Website](https://missouriavenuebaptistchurch.com/admin) (the admin site saved in Favorites bar)
    2. Login as ***mabcadmin***
    3. In the left sidebar, hover over ***Sermons***
    4. Select ***Add New Sermon*** and use these settings—
        - **Title** - \{ *Sermon Title* \}
        - **Description** - \{ *Verse Reference for Sermon* \}
        - **Video for Sermon (*Sermon URL*)** - \{ *the link copied from [here](#get-livestream-link), replacing `live` with `embed`* in the URL\}
    5. In the right sidebar, select the ***Sermon Speaker***
    6. Click ***Publish***

</Steps>

## OBS

<Steps>

    ### Send Video to TVs
    1. Open OBS
    2. Right-Click in the video panel
    3. Hover over ***Fullscreen Projector (Preview)***
    4. Select ***LCD TV***

    ### Select Livestream
    1. Click ***Manage Broadcast***
    2. Click the ***Select Existing Broadcast*** Tab
    3. Select the livestream for the upcoming service
    4. Click ***Select Broadcast***

</Steps>
