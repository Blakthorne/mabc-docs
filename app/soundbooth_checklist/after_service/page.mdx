# After Service

## Audacity Part I

1. Click the ***Stop*** button
2. Save the Audacity Projector

## Attendance

1. Set the ***Service Ended*** time (`Win` + `Shift` + `;`)
2. Save the file
3. Close the spreadsheet program

## Audacity Part II

<Steps>

    ### Export Sermon
    1. Highlight the audio between the ***In*** and ***Out*** markers
    2. Go to ***File*** > ***Export*** and use these settings—
        - **Filename** - `{ YYYYMMDD }-Sun{ AM/PM }-{ Speaker Last Name }` (e.g. `20250518-SunAM-Polar`)
        - **Destination Folder** - *`Desktop/`*
        - **Format** - `MP3`
        - **Channels** - `Stereo`
        - **Sample Rate** - `44.1kHz`
        - **Quality** - `64kbps`
        - **Bit Rate Mode** - `Constant`
        - **Export Range** - `Current Selection`
    3. Click ***Export***

    ### Export Full Audio Recording
    1. Click anywhere in the Audacity window to unselect the previous track Selection
    2. Go to ***File*** > ***Export*** and use the same settings as used [previously](#export-sermon) except for these—
        - **Filename** - `{ YYYY-MM-DD } Sun{ AM/PM }-{ Speaker Last Name }` (e.g. `2025-05-18 SunAM-Polar`)
        - **Destination Folder** - *`Sermon Archive/{ Current Year }`*
        - **Quality** - `128kbps`
        - **Export Range** - `Entire Project`
    3. Click ***Export***
    4. Close Audacity

</Steps>

## Website

1. Navigate to the [previously created sermon](/soundbooth_checklist/computer_preparation#add-sermon-page)
2. Click the ***Add Media*** button that's next to the ***Audio*** label
3. Click on the ***Upload Files*** tab
4. Go to the ***Desktop/*** Folder
5. Select the sermon audio file that was [previously exported](#export-sermon)
6. Once the file has uploaded, click ***Select***
7. In the right sidebard, click ***Update***

## Backup Files in Finder

<Steps>

    ### Audio
    1. Go to the ***Sermon Archive/\{ Current Year \}*** folder
    2. Copy the full service audio file that was [previously exported](#export-full-audio-recording)
    3. Go to the ***Backup Sermon Archive/\{ Current Year \}*** folder
    4. Paste the file

    ### Video
    1. Go to the ***Temp Video/*** folder
    2. Delete the ***.mkv*** file as it is no longer needed
    3. Copy the ***.mp4*** file from the service
    4. Go to the ***Backup Video Archive/\{ Current Year \}*** folder
    5. Paste the file
    6. Go back to the ***Temp Video/*** folder
    7. Delete the ***.mp4*** file

</Steps>
