# Setup for Next Service

<Callout type="info">
    We've found that it's better to complete these tasks directly after the previous service has ended in preparation for the next service. It just makes things easier when getting things ready during the week.
</Callout>

## OBS

1. Click ***Manage Broadcast***
2. Click on the ***Create New Broadcast*** button and use these settingsâ€”
    - **Title** - Sunday \{ Morning/Evening \} Service - \{ Full Month Name \} DD, YYYY (e.g. Sunday Morning Service - March 18, 2025)
    - **Description** - See **Appendix B** - add reference
    - **Privacy** - Public
    - **Category** - Nonprofits & Activism
    - **Is this video made for kids?** - No
    - **Thumbnail** - Leave blank
    - **Latency** - Normal
    - **Enable DVR** - Yes
    - **360 Video** - No
    - **Schedule for Later** - Yes
    - **Enable Auto-Start** - No
    - **Enable Auto-Stop** - No
    - **Datetime** - Date and time for the service
3. Click ***Schedule Broadcast***
4. Close OBS

## Website

<Steps>

    ### Get YouTube Livestream Link
    1. Go the the ***YouTube Studio Dashboard*** (saved in the Favorites bar)
    2. Click ***Content*** in the left sidebar
    3. Go to the ***Live*** tab
    4. Open the ***Upcoming*** livestream
    5. Copy the available link

    ### Update the Livestream Page of the Website
    1. On the Admin dashboard of the website, click ***Pages*** in the left sidebar
    2. Go the ***Livestream*** page
    3. Replace the URL with the one [just copied](#get-youtube-livestream-link) from ***YouTube***
    4. Replace `live` with `embed` in the URL
    5. Click ***Update***

</Steps>
